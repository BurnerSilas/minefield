
# Minefield

Minefield is a project for the ESP-32.

A digital audio effect panel.

The basic general idea is...

*Analog Audio Input* -> ADC -> *Digital Input* -> ESP-32 -> *Digital Output* -> DAC -> *Analog Audio*
